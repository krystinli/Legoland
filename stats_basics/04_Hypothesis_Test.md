## Selecting a Hypothesis Test
`Discrete` data: ie, tossing a coin (head/tail)
  - Do you have an avg rate? Yes => **Poisson**
  - Else:
    - Sample size: if N<50 => **Binomail**
    - Sample size: if N>=50:
      - If p < 0.1 => **Poisson**
      - If p >= 0.1, normal approximation to the binomail distribution
